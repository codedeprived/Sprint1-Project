# Food Plaza Online

Welcome to Food Plaza Online! This project aims to create an online platform for managing food orders and deliveries efficiently. Below are the key details about the project:

## Technologies Used

- Developed using **Core Java** for the backend logic.
- Utilizes **JDBC (Java Database Connectivity)** for database operations.
- **SQL (Structured Query Language)** for managing the database schema and queries.
- **Object-Oriented Programming (OOPS)** principles are followed for designing classes and relationships.
- **MVC (Model-View-Controller)** architecture is implemented to ensure separation of concerns and modular development.

- **User Layers**:

## How to Use

1. **Setup**:
   - Ensure you have Java JDK installed.
   - Set up a MySQL database instance (locally or on a server) and import the SQL scripts provided in the `sql Scripts` directory.

2. **Configuration**:
   - Modify the JDBC connection settings in `src/main/java/Utility/DbConnection.java` to match your database credentials.

3. **Build and Run**:
   - Compile the Java files using `javac` or use an IDE like IntelliJ IDEA, Eclipse, or NetBeans.
   - Run the application from your IDE or via command line using `java`.

4. **Usage**:
   - Follow the prompts to navigate through the user interface (CLI or GUI, depending on implementation).
   - Register as a new user, log in to your account, browse food items, place orders, and manage your profile.
